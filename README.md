# HelloWorld Java Program

This is a simple Java program that prints "Hello, World!" to the console.

## How to Run

1. Open a terminal and navigate to the `java-1` directory.
2. Compile the program:
   ```shell
   javac HelloWorld.java
   ```
3. Run the program:
   ```shell
   java HelloWorld
   ```

## Output
```
Hello, World!
```
